## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/durkasrimani/C-Language.git
```

2. Compile any C file

```bash
gcc filename.c -o output
```

3. Run the program

**Windows**

```bash
output.exe
```

**Linux / macOS**

```bash
./output
```
